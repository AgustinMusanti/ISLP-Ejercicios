# ISLP — Ejercicios
 
Resolución de los ejercicios del libro **An Introduction to Statistical Learning with Applications in Python (ISLP)**, que estoy usando para aprender machine learning y aprendizaje estadístico en Python.
 
La idea de este repo es dejar registro de mi avance por el libro a medida que lo voy leyendo: tanto los ejercicios de teoría como los de aplicación práctica con datos.
 
## Organización
 
El repo está dividido en dos carpetas:
 
- **`Ejercicios conceptuales/`** — Ejercicios teóricos: definiciones, comparaciones de métodos, trade-off sesgo-varianza, KNN a mano, etc.
- **`Ejercicios aplicados/`** — Ejercicios prácticos con código en Python (notebooks de Jupyter)
 
## Sobre el libro
 
*An Introduction to Statistical Learning* (Gareth James, Daniela Witten, Trevor Hastie, Robert Tibshirani, Jonathan Taylor). Edición en Python, 2023.
 
- Sitio oficial: https://www.statlearning.com/
- PDF gratuito: https://hastie.su.domains/ISLP/ISLP_website.pdf
- Paquete `ISLP` (datasets y funciones del libro): https://islp.readthedocs.io/
## Stack
 
Python · NumPy · pandas · matplotlib · scikit-learn · paquete `ISLP`
